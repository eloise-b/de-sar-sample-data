This folder contains the following html pages:

* [01_product_info.html](01_product_info.html) - high-level information about the product, including data availability and metadata
* [02_demonstration.html](02_demonstration.html) - a simple demonstration of key steps involved in loading and preparing SAR backscatter data for analysis
* [03_loading_with_stac.html](03_loading_with_stac.html) - more detailed information of how to work with the STAC API
* [04_loading_with_datacube.html](04_loading_with_datacube.html) - more detailed information of how to work with the Datacube API
* [05_transforming_data.html](05_transforming_data.html) - more detailed information about transformations such as masking, speckle filtering, and converting to different backscatter normalisation conventions.
