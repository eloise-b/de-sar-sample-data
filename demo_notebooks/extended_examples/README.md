# Extended Example Notebooks

These notebooks are designed to demonstrate additional capability that may be useful to you.

These notebooks focus on using odc-stac, and as such, may need to be adapted to work in the DEA Sandbox.

* [dataloading_known_scenes.ipynb](dataloading_known_scenes.ipynb) - demonstration of a utility function that can be used to search for and download the GeoTIFF for a known scene ID. 
* [visualisation_sar_optical_timeseries_animation.ipynb](visualisation_sar_optical_timeseries_animation.ipynb) - demonstration of how to load SAR imagery over time and produce an animation. Also includes a demonstration of how to load optical imagery for the same area and produce a time series animation comparing optical and SAR.
